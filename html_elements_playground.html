üìö Cheatsheet of Common HTML Elements

1. Structural Elements

  • <html>, <head>, <body>
  • <header>, <footer>, <main>, <section>, <article>, <aside>, <nav>

2. Text Elements

  • <h1> to <h6>, <p>, <span>, <strong>, <em>, <br>, <hr>, <blockquote>

  • <img>, <video>, <audio>, <iframe>, <canvas>, <figure>, <figcaption>

4. Form & Interactive

  • <a>, <button>, <form>, <input>, <select>, <label>, <textarea>

  • <div>, <ul>, <ol>, <li>

üß© Categorization of Elements

Empty (Void) Elements ‚Äî Only Opening Tag

  • <br>, <hr>, <img>, <input>, <link>, <meta>, <source>, <area>, <base>, <embed>, <track>, <wbr>

Normal (Container) Elements ‚Äî Opening and Closing Tags

  • <p>...</p>, <div>...</div>, <h1>...</h1>, etc.

Semantic vs Non-Semantic Elements

  • Semantic: <article>, <header>, <footer>, <nav>, <section>, <main>
  • Non-Semantic: <div>, <span>

Interactive Content

  • <a>, <button>, <details>, <input>, <label>, <select>, <textarea>

Metadata Content (head only)

  • <meta>, <base>, <title>, <link>, <style>, <script>

üß† HTML Theory

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses elements or "tags" to structure content like text, images, videos, and forms.

Tags usually come in pairs: an opening tag <tag> and a closing tag </tag>. Some tags, called void elements, don‚Äôt need a closing tag (e.g., <br>, <img>).

Tag Description Type
<p> Defines a paragraph Block
<span> Inline text container Inline
<div> Generic container Block
<br> Line break Void / Inline
<img> Embeds an image Void / Inline
